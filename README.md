📌 Project Description

The Online Ticket Booking System is designed to demonstrate the working of a basic reservation system. The program tracks the availability of seats for an event and allows users to interact through a menu-driven interface.

The system updates seat availability dynamically whenever a booking or cancellation occurs.

This project helps in understanding core C++ programming concepts and how they are applied in real-world systems.

⚙️ How the Program Works

The program starts by displaying the total number of available seats.

A menu is displayed to the user.

The user selects an option from the menu.

Based on the selection:

Ticket is booked

Ticket is cancelled

Seat availability is displayed

The system updates the number of remaining seats.

The process continues until the user chooses Exit.
